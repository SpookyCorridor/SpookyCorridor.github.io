Larvae.

A new way to consume and produce long-form content.

Process

Building Larvae was my way of learning the PHP framework Laravel, which is very similar and inspired by Ruby on Rails. Going through this project has given me a better understanding of the "magic" that happens behind the scenes in both Rails and Laravel. Specifically, things like middleware setup, how they handle many-to-many relationships, and service providers.

I think my biggest hurdle on this project was actually managing to get it properly deployed somewhere. Most tutorials are for outdated versions of Laravel which led to a long night of troubleshooting. Now that I've figured it out though, it's pretty easy to manage.
