Stack Interview.

The latest web developer interview questions all in one place.

Process

Stack Interview is an API for the latest front-end and back-end interview questions being asked by employers around the world. It's hard to keep up on technologies you may be learning or need to know with how fast this industry moves. I wanted to create a resource that would allow other developers to take the database of questions here and either add to it or create their own platform to meet their needs. I also created a quiz game to show what could be done with the API for anyone who just wants to play around with it.

For this project I had to really think about how to build up the database and API. I wanted to keep it very streamlined and easy to work with. In order to do this I added only what is essential: the questions and basic stats about them like criteria, whether its a frontend or backend question, and how well it is recieved by the community. The API itself lets you do some useful queries like searching for certain keywords or ratings. Getting a key is as easy as signing up for Stack Interview.
